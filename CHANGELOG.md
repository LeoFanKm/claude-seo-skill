# Changelog

All notable changes to claude-seo-skill are documented here.

## [4.0.0] — 2026-03-13

### Added
- **seo-keyword-research**: Complete keyword research methodology — seed expansion, intent classification, topic clusters, competitor gap analysis, priority scoring formula
- **seo-competitor-analysis**: Full competitor SEO intelligence — keyword overlap matrix, Share of Voice, content strategy reverse-engineering, SWOT framework
- **seo-local**: Local SEO — GBP optimization checklist, NAP consistency audit, review management, LocalBusiness schema, quality gates for location pages
- **seo-audit**: +80 granular audit rules (7 categories × rule ID / pass condition / severity): Core SEO (CS01–CS15), Performance (PF01–PF12), Links (LK01–LK12), Images (IM01–IM10), Security (SC01–SC10), AI/GEO Readiness (AI01–AI10), Content Quality (CQ01–CQ11)
- **seo-geo**: Princeton KDD 2024 GEO research foundation — 9 optimization strategies with measured effectiveness percentages, GEO scoring formula
- **references/keyword-difficulty.md**: KD score interpretation table (5 tiers), industry benchmarks, search intent CTR data, DataForSEO Keywords API reference

### Updated
- Orchestrator (seo/SKILL.md): 22 → 25 sub-skills, 12 → 15 subagents, +3 Quick Reference commands
